# よくある会話

クライマー同士で何を話しているか。ジムのフロアで、外岩で、更衣室で、よくなされている会話についてまとめる。
観察とインタビューから見えてきた会話のパターン。

<div class="phrase-section">

## 応援

### 「がんば！」 <span class="easy">気軽</span>

頑張っている人を応援するときのセリフ。come on! と言う人も。

### ナイスです <span class="easy">気軽</span>

課題を完登した人に対してかける声。

### ないすぅ <span class="normal">親しい</span>

課題を完登した人に対してかける声。

### 強い <span class="normal">親しい</span>

課題の核心など、ハードな箇所を通過したときにかける言葉。
なぜか、あまり格上の人に使うのは躊躇われる。

### うまい <span class="normal">親しい</span>

他の人がやらないムーブで課題を進めたとき、バランスが必要な箇所を進めたときにかける言葉。
「強い」などと同様、なぜかあまり格上の人に使うのは躊躇われる。

## 登りかたについて

### もう少し左 <span class="easy">気軽</span>

直接確認できないフットホールドを探している人に、外から位置を教えるときの声かけ。

### この課題スタートどれだっけ <span class="easy">気軽</span>

外岩では口頭の引き継ぎが重要で、「あのスタートは右手で持つと安定する」のような局所的な知識が蓄積・伝承される。

### 100乗らない <span class="normal">親しい</span>

フットホールドに体重を100%かけないムーブの説明。小さいスタンスや、次の動作のために重心を逃がしたいとき、ベータを共有する文脈でよく出てくる言葉。

### 悪い <span class="solo">独り言</span>

持ちにくい・立ちにくいホールドを指す形容。「このカチ、悪い」「スタートホールド悪すぎる」のように使う。クライマーの文脈では良し悪しの価値判断ではなく、グリップのしにくさを指す技術的な言葉。

### デッド気味で <span class="normal">親しい</span>

ムーブのアドバイスで使う表現。「デッド」とは動きの頂点でホールドを取りにいくダイナミックなムーブのこと。「デッド気味で」は、完全に飛ぶほどではないが、少し動きを出して取りにいくニュアンス。ベータを伝えるときに口をついて出る。

### 足上げれる？ <span class="easy">気軽</span>

ハイステップや高い位置にあるフットホールドに足を上げられるかを確認する言葉。柔軟性や股関節の可動域によって踏めるかどうかが変わる場面で、ベータ共有の入口として出てくる。「上げれるなら、そこに乗れると楽になる」と続くことが多い。

### スタで届くんだ？ <span class="easy">気軽</span>

ダイナミックなムーブが必要と思われていた箇所を、静的な動き（スタティック）でホールドに届かせているのを見て出る言葉。リーチや体型の違い、姿勢の工夫によって同じ課題でもムーブが変わる、クライミングの面白さが凝縮された一言。

### レイバック気味で <span class="normal">親しい</span>

縦ホールドや岩の稜線を引っ張りながら、足で押し返すレイバックムーブを使うよう促すアドバイス。「完全にレイバックじゃなくていいけど、少し引いて」というニュアンスで使われる。

### 正対で <span class="normal">親しい</span>

壁に対して体を真正面に向けた姿勢のこと。ヒールを効かせたり体を開いたりせず、真っ直ぐ向いて登るよう伝えるときに使う。「正対の方が安定する」「正対で取りにいって」のように、ベータの一部として出てくる。

### バラシではできてる <span class="easy">気軽</span>

課題を区間に分けてそれぞれ練習すると全部できるのに、通しで登ると繋がらないときの言葉。「バラす」とは課題を部分ごとに分解して練習すること。完登まであと一歩のフラストレーションを表すと同時に、「ムーブは解決している」という手応えも含んでいる。

### バラしてみたら？ <span class="easy">気軽</span>

通しで登ろうとして詰まっているときに、区間ごとに分けて練習することを勧める言葉。「まずムーブを解決してから繋げよう」という提案で、行き詰まりを打開するアドバイスとしてよく出てくる。

### あぁ、バラシですか <span class="easy">気軽</span>

完登に見えたのに本人がそれほど喜んでいないとき、通しではなく区間練習（バラシ）だったのかを確認する質問。「おめでとう」と言いかけて、様子を見て「もしかして…？」と切り替わる瞬間に出る。

### スタートできない <span class="solo">独り言</span>

課題の最初のムーブが起こせない、あるいはスタートホールドに体が浮かない状態。グレードを問わず、自分のレベルを大きく超えた課題や、特定のムーブが苦手な課題でよく漏れる言葉。

### パンプしてる <span class="solo">独り言</span>

前腕に乳酸が溜まり、握力が急激に落ちていく感覚を指す言葉。リードの核心手前やロングルートの途中で出る。「腕が張る」「前腕がパンパン」とも言う。パンプすると指が開き始め、ホールドを保持できなくなる。

### シェイクして <span class="easy">気軽</span>

パンプした前腕を回復させるため、腕をぶらぶら振るよう外から促す言葉。レストポジション（安定して片手を離せる箇所）を見つけたときに、外で見ている人が声をかける。シェイクで乳酸の抜けが早まる。

### フリクションない <span class="solo">独り言</span>

ホールドやスラブ面に足や手が滑り、摩擦が効かないと感じたときに漏れる言葉。シューズのラバーが消耗しているとき、岩が湿っているとき、ホールドに汗や粉チョークが溜まっているときに出る。

### ヌメってる <span class="solo">独り言</span>

ホールドの表面が汗や湿気でぬるぬると滑る状態を指す言葉。「フリクションない」がグリップ力の不足を指すのに対し、「ヌメってる」はホールドそのものが脂っぽく湿ってべたつく感触を指す。気温が高い日や、人がたくさんトライした後に出やすい。

### フリクションある内に落としたい <span class="easy">気軽</span>

外岩で時間が経つにつれて岩の温度が上がったり湿度が変わったりして摩擦が落ちていくとき、まだコンディションがいい早い時間帯に課題を完登したいという気持ちを表す言葉。

### 今何個？ <span class="easy">気軽</span>

リードで、クライマーが何本目のボルトまでクリップしたかをビレイヤーや周囲が確認する言葉。ルートの進捗を把握するための声かけで、核心の手前で止まっているときや、テンションをかけてから再トライを始めるときに交わされる。

### テンション <span class="easy">気軽</span>

リードクライミングで、ロープを張ってもらいたいときにビレイヤーへ向けて叫ぶ言葉。手が限界のとき、一時停止して次の手を考えたいときに出る。英語の "take!" に相当する。

### 降ろして <span class="easy">気軽</span>

リードで登り終えたあと、またはトライを切り上げるときにビレイヤーへ告げる言葉。「ロワーダウン」と言う人もいる。登り手が合図するまでビレイヤーは待つため、この一言が動作の起点になる。

### マッチ！ <span class="easy">気軽</span>

課題の途中、あるホールドで両手を揃えると次のムーブが楽になる場面で、外から飛ぶベータの声かけ。「そこでマッチしてから次を取りにいくといい」という情報を一言に圧縮した形。

### ゴールちゃんとマッチして <span class="easy">気軽</span>

ゴールホールドに両手を揃えて触れることがコンプリートの条件になっている課題で、外から声をかけるときの言葉。マッチとは両手を同じホールドに合わせること。本人は必死でゴールを掴んだ瞬間に忘れがちで、周囲が思わず叫ぶ。

## グレードの話

### あの◯段、触った？ <span class="normal">親しい</span>

特定のグレード帯の課題を試したかどうか聞く定番の話題。

### グレード変わってるんだけど <span class="easy">気軽</span>

ホールド替えや再設定でグレードが変更されたとき、あるいは感じているグレードと表記が合わないときに出る言葉。「前より難しくなってる」「これ絶対◯級じゃない」という不満が混じることも多い。

### このジム辛い <span class="easy">気軽</span>

グレード設定が厳しく、表記より実際の難易度が高いと感じるときの言葉。クライミングの「辛い（からい）」はグレードが甘くない＝シビアという意味で使われる。ホームジムと別のジムを掛け持ちするクライマーがよく口にする。
## 怪我の話

### 指皮なくなった <span class="easy">気軽</span>

課題をたくさんトライして、指先の皮が削れてしまったときに口から出るひとこと。
登り込んでいる証拠でもあり、「今日はそろそろ終わりかな」という区切りの合図にもなる。

### ぱきった <span class="solo">独り言</span>

指の腱鞘や滑車（プーリー）を痛めた瞬間に出る言葉。「パキッ」という感触や音からきたスラング。
軽度のものから全治数か月になるものまで幅があるが、クライマーの間では日常的に飛び交う言葉。「またぱきった」と話す人も少なくない。

## ジム・ホールドの話

### 磨いたらいけます <span class="easy">気軽</span>

惜しいところまで来ているとき、ブラシでホールドの古いチョークを落とせば完登できるのではと提案する言葉。「もうちょっとでいける」という観察とセットで出てくる。外岩では岩の表面を磨いてフリクションを回復させることが一般的で、次のトライ前のルーティンになっている。

### 今合計何個？ <span class="normal">親しい</span>

セッション中に今日何課題完登したかを確認する言葉。コンペ形式の練習や、目標を決めて登っているときに出てくる。「もう10個いった？」のように自然に進捗を共有する流れになる。

### この壁、ホール替えいつだっけ <span class="easy">気軽</span>

落とせない課題がある人に対して、ホールド替えがある前に落とせるといいですねというニュアンスで切り出すことが多い。

### この課題◯◯さんのやつ <span class="normal">親しい</span>

課題を触った瞬間や見た瞬間に、セッターの癖を感じ取ったときに出る言葉。「やっぱり」がつくことも多く、名前を確認する前から「あの人のだ」とわかるほどセッターの個性が出ている課題で起きる。

### いつもどこで登ってますか？ <span class="easy">気軽</span>

普段見かけない人を見かけたときに聞く質問。
これを切り出す人は大体そのジムの常連。

## 外岩・遠征の話

### 外岩行くの？ <span class="easy">気軽</span>

ジムで会った相手が外岩に行く気配を見せたときに聞く一言。外岩に行かない人が外岩勢に聞くこともあれば、外岩勢同士の情報交換のきっかけになることもある。

### コンディション悪い <span class="easy">気軽</span>

雨上がりで岩が濡れていたり、気温が高く手汗でホールドが滑るときに出る言葉。外岩では天候・気温・湿度がパフォーマンスに直結するため、登る前から「今日はコンディションが…」と話題になる。登れない言い訳にもなる。

### 足場悪いから気をつけて <span class="easy">気軽</span>

外岩でのランディング（着地点）周辺が岩や根で不安定なとき、登る前やトライ中に周囲が声をかける言葉。落ちた瞬間に足をひねるリスクがあり、スポッターの立ち位置の確認と合わせて交わされる。

### スポットお願い <span class="normal">親しい</span>

落下時に体を支えてくれるスポッターを頼む言葉。外岩のハイボルダーや着地が悪い課題でトライする前に声をかける。スポットを入れるかどうかの判断自体がリスク認識の共有になる。

### マット失礼します <span class="easy">気軽</span>

外岩で他のパーティーのクラッシュパッドを使わせてもらうときの一言。マットは持ち込むのが大変なため、岩場では見知らぬ人同士でも貸し借りが自然に起きる。この一言が会話のきっかけになることも多い。

## 器具・ギアの話

### フューチェラ履いてた人ですよね <span class="easy">気軽</span>

シューズのモデルで相手を識別・記憶しているときの声かけ。クライマーはシューズに目がいくため、顔より靴で人を覚えていることがある。ギアが会話の入口になり、初対面でも距離が縮まるきっかけになる。

### いつも荻パンで登ってますよね <span class="easy">気軽</span>

ホームジムで人を覚えていて、別の場所で再会したときの声かけ。「荻パン」はB-PUMP荻窪の通称で、ジムの呼び名自体がコミュニティの共通言語になっている。シューズ同様、どこで登っているかがその人を特定する手がかりになる。

### 同じシューズですね <span class="easy">気軽</span>

他のクライマーが自分と同じモデルのシューズを履いているのに気づいたときの一言。それだけで会話が始まることがある。「どうですか？」「サイズ感どうしました？」と続き、ギアトークへの自然な入口になる。

### フューチェラ私も持ってます <span class="easy">気軽</span>

特定のシューズモデルへの共感を求める言葉。Five Tenのチーム5.10はコスパと履き心地で支持層が多く、名指しで語られやすい。「そのシューズどうですか？」より一歩踏み込んだ、すでに知っている前提の話しかけ。

完登動画や写真をInstagramに投稿しているか聞く言葉。
