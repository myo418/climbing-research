# グレード

## このページの目的

フリークライミングにおける**グレード（難易度体系）**を整理する。

- どんな体系があるのか（ボルダー／ルート、地域別）
- どう換算するのか
- グレードの持つ主観性・ブレ・文化的側面
- 研究テーマ（なぜリスクを受け入れてまで登るのか）との接続点

関連: [climbing-and-injury](../../climbing-and-injury/) / [genres](../genres/)

---

## 1. グレードとは

- ルートや課題の**難易度**を示す記号体系
- 初登者が提案し、後続の登攀者の合意で収束していく（＝**コンセンサスグレード**）
- 絶対値ではなく、ジム／エリア／時代／セッターでブレる
- 「自分のグレード」を更新することが多くのクライマーのモチベーションの中核

## 2. ボルダリングのグレード

### 2-2. フォンテーヌブロー（Font）グレード

フランス・パリ近郊のフォンテーヌブロー発祥。ヨーロッパで主流。

- 表記: `4` `5` `5+` `6A` `6A+` `6B` `6B+` ... `8C+` `9A`
- 6A以上はA/B/Cの細分化と `+` の有無で1ランク=2段階
- ルートのフレンチグレードと数字が被るので混乱しやすい → 頭に `f` や "Font" を付けて区別

### 2-3. 段級（日本・アジア）

柔道・将棋から借りた表記。岩場ごとのローカル性が強い。

- 級: **10級 → 1級**（数字が小さいほど上）
- 段: **初段 → 二段 → 三段 → 四段 → 五段 → 六段**（数字が大きいほど上）
- 実運用では六段あたりが天井。それ以上の難度はVスケール／Fontで表記されることが多い
- 1級と初段の境界が最も議論される
- 岩場（御岳、瑞牆、小川山…）で基準が異なる

## 3. ルート（リード／スポート）のグレード

### 3-1. YDS（ヨセミテ・デシマル・システム、アメリカ）

| 区分 | 内容 |
|---|---|
| クラス1〜4 | ハイキング〜スクランブル |
| クラス5 | ロープを使う自由登攀 |
| 5.0 〜 5.9 | 初級〜中級 |
| 5.10a/b/c/d | 以降 `a/b/c/d` で細分化 |
| 5.11 〜 5.15d | 現行最高は5.15d |

- 5.10以上は `a/b/c/d` の4段階
- **R / X** 付き: プロテクションが乏しく落ちると危険／極めて危険
- 例: 5.11a R = 技術的には5.11a、だが落ちたら怪我の可能性

### 3-2. フレンチグレード（スポートの事実上の世界標準）

- 表記: `5a` `5b` `5c` `6a` `6a+` `6b` `6b+` ... `9a` `9b` `9b+` `9c`
- 6a以上は数字+a/b/c+`+`の有無で細分化
- 現行最高: **9c**（Adam Ondra "Silence"、2017）

### 3-3. UIAA（ドイツ語圏・中欧）

- ローマ数字: `I` `II` `III` ... `XII+`
- 各ランクに `-` `0` `+` の3段階
- アルプスの伝統地域で使用

### 3-4. 英国式（トラッドの本場）

**2軸のグレード**を並べる独自体系:

- **Adjectival（総合難易度＋危険度）**: M → D → VD → S → HS → VS → HVS → E1 → E2 → ... → E11
- **Technical（技術難易度）**: 4a / 4b / 5a / 5b / 6a / 6b ... 7b

例: `E5 6a` — 総合E5（かなり危険）、技術6a

プロテクションの得やすさ・ランナウト距離・墜落の深刻さを反映。**トラッド文化と危険の内在化**を色濃く示す体系。

### 3-5. 日本の段級（ルート）

岩場では段級がルートにも使われる。ボルダーと同じ表記だが難易度水準は別物（同じ「初段」でもルートとボルダーでは絶対難易度が違う）。

## 5. 登り方の区分（グレードと不可分）

同じグレードでも「どう登ったか」で価値が変わる。

- **オンサイト（OS）**: 初見で無落下、事前情報なし
- **フラッシュ（FL）**: 初見で無落下、事前情報あり（ムーブを聞いた等）
- **レッドポイント（RP）**: 練習後、無落下で完登
- **ピンクポイント（PP）**: 事前にヌンチャクがかかった状態でのRP（死語気味）
- **トップロープ（TR）**: 上からロープで確保。グレードは主張しないのが一般的

難しいグレードほどOS → FL → RPの差が大きい（例: 9aをRPできてもOSできるのは世界で数人）。

## 6. グレードの主観性・ブレ要因

- **身長・リーチ**: 届くか届かないかで2〜3ランク差
- **体型・柔軟性**: ヒールの効き、開脚度
- **得意タイプ**: スラブ／垂壁／かぶり／クラック／スローパー／ピンチ／カチ
- **岩質**: 花崗岩／石灰岩／チャート／砂岩／チョコストーン
- **エリアの基準**: 「辛い（サンドバッグ）／甘い（ソフト）」が岩場ごとにある
- **ジムとの乖離**: ジム初段 ≠ 岩場初段（ジムの方が甘い傾向）
- **時代**: 昔の「初段」が今の「三段」に再評価されることも

## 7. 歴史と最高グレードの変遷

### ボルダー最高グレードの変遷
- 2000年頃: V14（8B+）が天井
- 2002: V15（Dreamtime / Fred Nicole）
- 2011: V16（The Story of Two Worlds / Dave Graham ら）
- 2016: **V17**（Burden of Dreams / Nalle Hukkataival、フィンランド）
- 2023-2025: 複数のV17課題が成立、V18提案も出始める

### スポート最高グレードの変遷
- 2001: 9a+（Biographie / Chris Sharma）
- 2008: 9b（Jumbo Love / Sharma）
- 2012: 9b+（Change / Adam Ondra）
- 2017: **9c**（Silence / Adam Ondra、ノルウェー）
- 以後、9cは複数本に増えつつある

### 日本
- 平山ユージ、小山田大、安間佐千、楢崎智亜、野中生萌、伊藤ふたばらが世界水準で活動
- ボルダー日本国内の最高グレードは五段+〜六段相当まで

## 8. グレードと研究テーマの接続

### なぜグレードが怪我のリスクを増幅するのか
- **グレード更新圧**: 自分の限界を更新し続けたい欲求 → 過負荷・休息不足
- **比較可能性**: SNS時代にグレードは「数字で見える自分の価値」になりやすい
- **サンドバッグ問題**: 「辛い岩場の初段」を登ることが誇りになり、力量を超えたトライを生む
- **ボルダーは着地リスク**: 高グレード=ハイボルダー化しやすく、落下時の怪我リスクが急上昇

### グレードはクライマーにとって何か
- 自己の身体能力の**座標系**
- 他者と共有できる**言語**（「あのルート5.12b」で会話が成立）
- 更新する対象としての**物語装置**
- だが、**グレード病**（グレードのみを追う状態）への批判も根強い

→ インタビューで「あなたにとってグレードとは？」を聞くと、動機構造が見えやすい。
