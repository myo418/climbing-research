# インタビュー方法

## このページの目的

本リサーチの中心的な一次情報源は**クライマー本人の語り**。
そこから「なぜクライミングが好きになり、のめり込み、怪我のリスクがあっても続けているのか」を汲み取るための**プロセス設計**をここにまとめる。

関連: [climbing-and-injury](../../climbing-and-injury/) / [趣味クライマーインタビュー](../../climbers/interview/)

---

## 1. 設計思想

### 1-1. 「なぜ？」を直接聞かない

「なぜ続けているんですか？」と直接聞くと、本人が自分でも言語化できていない動機は**既製品の答え**（「楽しいから」「達成感があるから」）に収束してしまう。

→ **具体的な体験**を語ってもらい、その語り方・選ぶ言葉・間・矛盾から動機を浮かび上がらせる。

### 1-2. 半構造化インタビュー

- 質問の骨子は用意する（後述）
- 順番・深掘りは相手の反応で変える
- 沈黙を恐れない（本人が考えている時間）
- 相手の言葉をオウム返しで返して、さらに語らせる

### 1-3. 話の順序

感情の深い話は後半。序盤で信頼関係を作ってから核心に入る。

```
挨拶・自己紹介
  ↓
入口（どう出会ったか） ← 話しやすい
  ↓
のめり込み（日常の変化）
  ↓
身体・技術（得意／苦手）
  ↓
怪我・恐怖（感情が動く）
  ↓
続ける理由（核心）
  ↓
振り返り（話してみて何を感じたか）
```

## 2. 質問リスト

### 入口 — どう出会ったか
- 初めてクライミングに触れたのはいつ、どんな状況で？
- 最初のセッションで何を感じた？（身体感覚・感情）
- また行こうと思ったきっかけは？
- 初めて「登れた」と手応えを感じた瞬間は覚えている？

### のめり込み — 日常の変化
- 頻度はどう変化していった？
- ジムから岩への移行はあった？何がきっかけ？
- 生活の中でクライミングの優先順位はどこ？
- お金・時間・人間関係はどう変わった？
- 始める前の自分と比べて、変わったと思うところは？

### 身体・技術
- 自分の登りで大事にしていることは？
- 「うまくなった」と感じた瞬間は？
- 壁にぶつかった時期はあった？
- 身体は変わった？（筋肉、指の肥大、怪我跡）
- グレードとどう付き合っている？

### 怪我・恐怖
- これまでで一番大きな怪我は？（具体的に）
- そのとき「やめよう」と思った？なぜ／なぜそう思わなかった？
- 復帰してから登りは変わった？（慎重／むしろ攻める）
- 周囲から「危ない」と言われたことは？どう返している？
- 「怖い」と「楽しい」はどう同居している？

### 続ける理由（核心）
- クライミングをやめるとしたらどんな時？
- 80歳まで登っている自分、想像できる？
- クライミングがなかったら自分は何を失う？
- 他人にこの魅力を説明するとしたらどう言う？
- 言葉にしづらい部分があるとしたら、それは何？

### 振り返り
- 今日話してみて、自分でも意外だったことは？
- 聞かれたかった質問・まだ話したいことは？

## 3. 対象者の想定

多様なフェーズ／背景の声を集めることで、継続／離脱の境界が見えてくる。

| 区分 | 着眼点 |
|---|---|
| 入門〜1年未満 | 何に惹きつけられているのか、最初期の感情 |
| 中級（1〜5年） | 習慣化のメカニズム、最初のグレード壁 |
| 長期継続（10年以上） | 生活との統合、価値観の変遷 |
| 離脱経験者（やめて戻った） | 離れていた期間の喪失感、戻った動機 |
| 引退者（怪我・その他で） | やめる側から見えるクライミング |
| 家族・パートナー | 当事者から見えない視点（心配・葛藤） |

※ プロ／セミプロは[有名クライマー](../../climbers/famous/)で別枠扱い。ここでは**趣味クライマー**を主対象とする。

## 4. 記録方法

### 4-1. 音声（必須）

- **メイン**: iPhone / IC レコーダーで録音
- **バックアップ**: もう一台のスマホでも同時録音（録音失敗は致命傷）
- **開始前に必ず**: 録音許可と録音中であることを口頭確認 → 録音内に残す

### 4-2. 映像（同意があれば）

- 固定カメラで表情・身振りを記録
- インタビュー中の「間」「視線」「身体の動き」は言葉以上に情報量がある
- 後日編集して本人確認を取れば、公開動画にも使える

### 4-3. 手書きメモ（最小限）

- キーワード、印象的な表現、沈黙の長さ
- 書きすぎると相手を不安にさせるので要所のみ
- インタビュー直後に**フィールドノート**として清書する（鮮度が命）

### 4-4. 後処理

1. 文字起こし（Whisper等で自動化 → 手で整形）
2. タグ付け（入口／のめり込み／身体／怪我／継続動機／矛盾）
3. 引用候補の抽出（短く印象的な発言）
4. 非言語情報の注記（「ここで長い沈黙」「笑いながら」など）
5. 本人に確認を取って公開版を作成

## 5. 事前準備

### 5-1. 相手について調べる
- SNS、ブログ、登攀歴
- 共通の知人・岩場
- 「下調べしてきました」と伝えると信頼が上がる

### 5-2. 場所の選び方
- 本人が落ち着く場所（ジム・カフェ・自宅）
- 騒音の少ない環境（録音品質）
- プライベートな話題も出せる距離感

### 5-3. 同意書
- 録音・録画の可否
- 公開範囲（内部サイトのみ／外部公開）
- 実名／匿名／仮名の選択
- 「この部分はカットしてほしい」と後から言える仕組み

### 5-4. 持ち物チェックリスト
- [ ] 録音機 ×2
- [ ] 予備バッテリー／モバイルバッテリー
- [ ] 同意書（紙）
- [ ] 名刺
- [ ] メモ帳・ペン
- [ ] 質問リスト（手元参照用）
- [ ] お礼の品（必要に応じて）

## 6. 当日の流れ

```
1. 挨拶・アイスブレイク（5分）
   └ 名刺を渡す

2. 研究の説明（5分）
   ├ テーマ：なぜクライミングを続けているのか
   ├ 武蔵野美術大学4年 後藤ゼミ
   └ 今日聞きたいことのテーマだけ伝える（詳細な質問は出さない）

3. リサーチサイト・SNSの共有（2分）
   └ QRコード or リンクを見せる

4. 同意確認・録音開始（3分）

5. インタビュー本編（60〜90分）

6. 振り返り・御礼（5分）
   ├ 話してみての感想を聞く
   └ 後日連絡の方法を確認
```

- 研究テーマ（一行）：「クライミングと怪我、続ける理由」
- メール・Instagram ID

- リサーチ用Instagram QRコード
- 「インタビュー協力者を募集中」の一行

→ 渡した瞬間に「この人何やってる人か」が伝わるのが目的。

## 8. リサーチ用SNS・サイトの運用

### 8-1. なぜ持つか
- **信頼の担保**: 「こういう活動をしている」が一目で伝わる
- **継続的な関係づくり**: インタビュー後もフォローし合える
- **被験者からの逆アプローチ**: 「自分も話したい」と言ってもらえる
- **記録の公開**: 週次レポートや進捗が蓄積されていく

### 8-2. アカウント構成案

| 媒体 | 用途 |
|---|---|
| Instagram | クライマーコミュニティが集まる中心。写真＋短文で週次進捗 |
| リサーチサイト（内部） | 全インタビュー記録・ローデータ・分析 |
| リサーチサイト（外部公開） | 精査済みの成果、インタビュー協力募集 |

### 8-3. 発信する内容
- 週次レポート（進捗・考えていること）
- インタビューのハイライト（本人許可済み）
- 読んだ本・見た映画のメモ
- 岩場・ジム訪問記

### 8-4. インタビュー時の共有フロー
1. 名刺裏のQRから見てもらう
2. 「こういう形で発信します」とトーンを伝える
3. 公開範囲の希望を聞く（本人のどの情報を／どこまで出すか）
4. インタビュー後にDMで「ありがとうございました」＋関連投稿をシェア

---

## 9. 倫理・配慮

- **匿名化のオプション**を必ず提示。迷うなら匿名寄りで。
- 話したくない話題を**事前にヒアリング**（「触れてほしくないテーマはありますか」）
- 感情の動く話題（重大な怪我、仲間の事故、離脱）は**後半**に置く
- インタビュー後でも「**これは公開しないで**」と言える窓口（後日のメール・DM）
- 仲間の実名が出た場合は、その人にも確認が必要な場合がある
- 未成年へのインタビューは保護者同意

## 10. 分析・成果物への接続

### 10-1. 個別記録
- 内部サイト: 文字起こし全文＋メモ
- 外部サイト: 本人承認済みの要約＋引用

### 10-2. 横断分析
複数のインタビューから共通テーマを抽出：
- 入口パターン（誘われた／偶然／映像で知った…）
- のめり込みの引き金（達成／仲間／身体の変化…）
- 怪我体験と継続の関係
- 言語化しづらい動機の類型

→ [why-hypotheses](../../) 配下の分析ページに接続。

## 11. 未解決の問い

- 録音を意識すると「表向きの答え」になる問題の回避策は
- 複数回インタビューすべきか（初回は表層、2回目以降で深層）
- 本人確認のステップで「やっぱりこの話はなしで」となったときの扱い
- 定性データだけで十分か、アンケートで量的補強すべきか
- インタビュアー（自分）もクライマーである場合のバイアス
